🛠️ Requirements

Make sure you have the following installed:

Python 3.x

pip (Python package manager)


Install Dependencies

Run the following command to install all required packages:

pip install -r requirements.txt

If you don’t have pip, install it first:

sudo apt install python3-pip -y  # For Debian/Ubuntu


---

🔧 Setup

1️⃣ Clone the repository
```sh
git clone https://github.com/Kazuha787/YoYo-Auto-Reff.git
cd YoYo-Auto-Reff
```

2️⃣ Install the dependencies
```sh
pip install -r requirements.txt
```

3️⃣ Edit config.py (if needed)

Open the config.py file and enter your OyaChat referral link


4️⃣ Run the bot
```sh
python bot.py
```

---

⚡ How It Works

1. The bot connects to OyaChat and automatically registers new accounts


2. It uses your referral link to earn points for every signup


3. You can run the bot repeatedly to generate unlimited referrals




---

🖥️ Running on a VPS

For 24/7 automation, run the script on a VPS:

nohup python bot.py &

Check logs:

tail -f nohup.out

Stop the bot:

pkill -f bot.py
